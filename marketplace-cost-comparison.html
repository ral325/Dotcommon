Model assumptions
Etsy fee structure
6.5% transaction fee + 3% + $0.25 Etsy Payments processing fee + $0.20 listing fee per item. Listing fee amortised per transaction assuming each listing sells once per listing period. At higher transaction sizes the listing fee becomes negligible.
eBay fee structure
13.25% final value fee (typical general merchandise, managed payments). $0.30 per order fixed fee. First 250 listings per month are free — listing fee modelled as $0 for active sellers.
Amazon fee structure
15% referral fee (general merchandise; varies 6–20% by category). $39.99/month Professional Seller subscription amortised across all transactions at current GMV ÷ transaction size. FBA fulfilment excluded — seller-fulfilled only.
Escrow gas
Two on-chain transactions per trade: lock on purchase, release on confirmation. L2 gas estimated $0.01–$0.06 per transaction on Optimism based on 2024–25 observed fees. Model uses midpoint ($0.035). Gas costs are expressed in USD — in practice they fluctuate with network congestion.
Listing registry gas
Each new or updated listing writes a CID pointer to the on-chain registry contract — a SSTORE operation, more expensive than a simple transfer. Estimated $0.02–$0.08 per write; model uses midpoint ($0.05). Assumed 2 writes per listing per year (creation + one update on average). Annual cost = listings × 2 × $0.05, amortised per transaction.
Reputation write gas
One on-chain write per completed transaction — buyer submits a signed rating stored in a mapping. Estimated $0.03–$0.10 per write on Optimism; model uses midpoint ($0.065). This is a direct per-transaction cost, not amortised.
The Graph query fee
The Graph Protocol charges query fees in GRT passed through to consumers. Estimated $0.0001–$0.001 per query for simple subgraph lookups. Each transaction assumed to involve 3 queries: listing lookup, seller profile, order status. Model uses midpoint $0.0005/query × 3 = $0.0015/transaction.
ZK attestation (TLSNotary)
Sellers re-attest identity on a cadence set by the slider (default: 2/yr). TLSNotary has no fixed pricing — compute cost borne by the network. Estimated $0.00–$0.30 per proof; model uses $0.20 as a pessimistic planning value. On-chain verification gas estimated $0.08–$0.60 per submission; model uses midpoint $0.34. Annual cost per seller = zkPerYear × ($0.20 + $0.34), amortised per transaction.
IPFS pinning cost
Persistent listing storage via Pinata or equivalent. Pinata Growth plan: ~$0.15/GB/month. A typical physical goods listing (3–5 images + JSON metadata) is estimated at 3–15MB → $0.0005–$0.002/listing/month. Slider default of $0.008 is a conservative planning value for larger image sets. Annual cost = listings × $/listing/mo × 12, amortised per transaction.
Kleros escalation fee
Tier 3 disputes escalated to Kleros incur a juror fee. Kleros court fees on Gnosis Chain: approximately $15–$60 per dispute depending on court and appeal rounds; model uses $30 as planning midpoint. Expected cost per transaction = (escalation rate ÷ 100) × $30. At default 0.2% rate this is $0.06/transaction — the largest single uncertainty driver at high dispute rates.
Privy wallet fee
Privy charges ~$0.05–$0.10 per active user per month for embedded wallet infrastructure. Model uses midpoint $0.075/user/month = $0.90/year. Amortised per transaction by dividing by txPerYear. Sellers using their own self-custody wallets avoid this cost entirely — it is part of the accessibility layer, not the base protocol.
Onboarding gas
One-time cost to initialise a seller's on-chain profile (registering with the registry contract). Estimated $0.10–$0.50 on Optimism; model uses midpoint $0.30. Amortised over seller's expected account lifetime (slider, default 4 years) × txPerYear. Becomes negligible for active sellers with long account lifetimes.
DAO governance gas
On-chain Aragon OSx votes consume gas per participating member. Estimated $0.30–$1.50 per vote per member; model uses midpoint $0.90. Annual governance gas per member = daoVotesPerYear × $0.90. Amortised per transaction. Default 18 votes/year ≈ roughly bi-weekly with some gaps. This assumes every member votes in every proposal — actual participation rate would lower this cost.
RPC / node infrastructure
Network-operated Alchemy/Infura account or self-hosted Optimism node. Monthly cost set by slider (default $400/mo = $4,800/yr). Divided equally across all network members → per-member annual cost, then divided by txPerYear → per-transaction cost. At 2,000 members and $400/mo this is $2.40/member/year — negligible at reasonable transaction volumes, visible for low-volume sellers.
Fiat ramp (double conversion)
Buyer converts fiat → DAI (on-ramp, 0.5–1.5% via Stripe); seller converts DAI → fiat (off-ramp, 0.5–1.5%). Model uses midpoint of 1.0% per leg = 2.0% total. ERC-4337 paymaster sponsored gas adds ~$0.006/tx. Sellers holding DAI avoid both ramp legs and pay the protocol-only rate.
What is not included
Shipping costs (identical across all platforms — excluded for comparability). Advertising / promoted listings. Returns and chargeback processing. Dotcommon mutual credit interest cost (separate model). Amazon FBA storage and fulfilment. Peer arbitration fees at Tier 2 (assumed absorbed into the dispute reserve). Token price volatility.
Non-recurring costs — excluded from model
NRE items not included in per-transaction cost model
These are one-time or infrequent capital costs that do not recur per transaction. They are real and must be planned for separately but are excluded to keep the running cost comparison clean.

Smart contract security audits — escrow, on-chain listing registry, reputation contract, and any credit pool contracts each require an independent audit. Typical range from a reputable firm (Trail of Bits, OpenZeppelin, Spearbit): $20,000–$80,000 per engagement. A full initial suite of 3–4 contracts: $60,000–$200,000. Re-audits required after any significant upgrade.

Legal entity and cooperative formation — establishing the legal structure (multi-stakeholder cooperative, LLC wrapper, or foundation), drafting member agreements, and obtaining any required money transmission or financial services licences. Estimated range: $15,000–$75,000 depending on jurisdiction, with ongoing compliance counsel at $5,000–$20,000/year thereafter.

The Graph subgraph development and deployment — indexing contract events into a queryable subgraph is a one-time engineering cost. Once deployed, the per-query fee is captured in the running model above.

TLSNotary notary server infrastructure setup — standing up the attestation server environment is a one-time engineering and infrastructure provisioning cost. Ongoing compute is captured by the per-proof cost in the model above.
